# 术语

主要记录编程中常用的技术术语。



## 数组相关

| 术语     | 解释                                                         | 备注 |
| -------- | ------------------------------------------------------------ | ---- |
| 遍历     | 取出数组中的每一个元素的过程叫遍历，通常使用循环语句或迭代器来完成。 |      |
| 冒泡排序 | 将数组中的元素按照从小到大或从大到小的顺序排列。             |      |
|          |                                                              |      |
